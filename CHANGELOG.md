# 📝 AskMAI 更新日志 (Changelog)

所有本项目的版本迭代和重要功能更新都会在此文件中记录。

---

## 🚀 [v1.1.0] - 2026-06-01

### 📂 WebView 多模态与文件上传支持
- **功能新增**：完美支持在 WebView 窗口中调用原生系统文件选择器、调用相机拍摄及从相册选择图片。
- **权限管理**：适配 Android/iOS 底层文件访问和相机调用权限申请，彻底解决在部分大模型网页端无法上传图片、文档进行多模态交互的痛点。

### 🔄 AI 标签页拖拽排序
- **交互重构**：在设置面板的 AI 标签列表中增加了拖拽把手图标（Drag Handle），支持长按上下拖拽动态调整标签页展示顺序。
- **状态持久化**：用户拖拽排序后的顺序会被自动持久化保存至 `SharedPreferences`，并在应用重启时自动应用于顶部标签栏和多模型提问分发顺序。

### ⚙️ 全新设置抽屉 (DraggableScrollableSheet)
- **UI 优化**：重构了设置面板，采用 `DraggableScrollableSheet` 实现优雅的抽屉式拖动交互。
- **高度限制**：限制面板的最大显示高度为屏幕的 2/3，顶部拖动把手固定不随内部内容滚动。
- **便捷手势**：支持滑动内容到底部后下滑直接顺畅关闭折叠，解决了旧版设置页面撑满屏幕或在小屏设备上遮挡“编辑 AI”等弹窗的问题。

### 🚥 智能状态指示灯系统
- **状态重构**：重构了标签栏的圆形状态指示灯系统，并接入 DOM Mutation Observer 网页元素监听：
  - `灰色 (Loading)`：网页正在加载资源。
  - `绿色 (Loaded)`：网页加载完成。
  - `绿色呼吸闪烁 (Active / Thinking)`：**核心特性**。实时监听网页 DOM 树变动，当 AI 正在打字输出、搜索引擎正在检索时呼吸闪烁，使用户能直观感知后台模型的工作状态。
  - `红色 (Error)`：网页加载失败。
- **快速超时与重试**：新增 12 秒网页加载超时快速失败机制，并增加了重试遮罩层，提示一键刷新。加载失败时自动追加 VPN/代理等配置排查建议。

### ⚡ 文本注入引擎升级与策略模式
- **策略模式注入**：设计并实现了 `InjectionStrategy` 接口，解耦了文本注入和自动化提交逻辑，支持 `GenericStrategy`（普通 DOM 查找）与 `ReactFiberStrategy`（React Fiber 注入）。
- **攻克 React 锁屏问题**：针对字节跳动豆包、阿里通义千问等基于 React / Next.js / Slate.js 深度定制的虚拟 DOM 文本输入框，直接从 React Fiber 树的 Node 节点获取 React 控制器和编辑器实例，强行写入并触发更新。
- **修复重复提交**：修复了在 Qwen/ChatGPT 等网站上因为传统 `.click()` 导致发送两次或文本写入后被框架自动清空的顽疾。

### 🛡️ 手势、导航与焦点保护
- **内页后退拦截**：重构系统物理返回键逻辑。在网页内浏览时，系统返回键会被智能路由给 WebView 活跃页面用于向后导航，而不会直接退出 App 或关闭当前会话。
- **双击返回优雅退出**：在主界面实现双击物理返回键退出机制。第一次按下时会显示一个高颜值的 66% 透明度毛玻璃气泡 Toast 提示“再按一次退出”，自适应系统深色/浅色主题。
- **网页滚动焦点保护**：引入键盘焦点和可见性管理机制，彻底解决了在用户滑动/滚动 WebView 网页时，系统意外收回焦点导致软键盘频繁闪退、关闭的问题。

### ✂️ 视口剪裁体验升级
- **临时禁用视口**：在左侧操作栏增加了“临时禁用视口裁剪”功能。开启后可临时恢复标准网页视口，以便查看/操作隐藏的侧边栏或完整网页。
- **布局微调**：将视口微调对话框 (`ViewportAdjustDialog`) 启动按钮等统一移至界面左侧布局，为中间主显示区腾出了更大的纯净对话视野。

### 📦 全局配置文件与动态加载
- **静态配置提取**：引入 `assets/app_config.json` 统一管理应用全局默认配置，包括：
  - 默认主题模式（自适应/浅色/深色）
  - 标题栏是否默认显示
  - 网页加载策略（并行 / 顺序）
  - 默认启用的 AI 标签列表
  - 项目 GitHub 仓库地址
- **动态版本与关于对话框**：从 `package_info_plus` 中动态读取应用版本号（例如 `1.1.0`），点击版本号会弹出带 GitHub 链接的关于对话框，方便用户一键直达源码仓库。

## 🚀 [v1.0.0] - 2026-05-29

### 🎨 现代化 UI 架构与主题
- **设计规范**：实现了全新的圆形设计语言与统一主题配置（`theme_config.dart`），界面高级现代。
- **框架搭建**：使用 Flutter 平台重构应用，采用 **MVVM (Model-View-ViewModel)** 架构，结合 **Provider** 完成状态解耦与数据绑定。

### 🚀 核心自动化与视口裁剪
- **一键并发广播**：在底部全局输入框中输入问题，通过 `Future.wait` 并发调度 JavaScript 脚本向所有已开启的 WebView 执行输入及点击提交事件。
- **XPath 配置**：支持从 `assets/site_config.json` 读取和管理 AI 模型，通过 F12 浏览器控制台获取 `inputXPath` 与 `submitXPath` 即可支持自定义扩展添加任何新的大模型。
- **视口裁剪系统**：通过 `ClipRect` + `OverflowBox` 实现了对普通 AI 套壳网页的多余边框剪切与缩放，将网页对话区域完美填充至屏幕中，获得媲美原生 App 的高纯净度体验。
- **会话与备份**：自动记忆已启用的标签；支持在设置中一键导出、导入所有 AI 配置，方便跨设备迁移。
